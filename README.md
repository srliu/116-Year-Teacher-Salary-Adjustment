# 116 年教師調薪試算與分析系統 (116-Year-Teacher-Salary-Adjustment)

這個專案是一個網頁型態的互動工具，旨在因應中華民國 116 年（2027 年）公立學校教師薪資調整政策，提供便利的薪資試算、級距對比以及相關數據的視覺化呈現。本專案同時作為「資料科學期末專題製作」的成果發表。

## 🌐 線上展示
本專案已透過 GitHub Pages 部署，您可以直接點擊下方連結使用：
👉 [點此前往線上試算系統](https://srliu.github.io/116-Year-Teacher-Salary-Adjustment/)

## ✨ 功能特點
* **互動式薪資試算：** 輸入現行的薪級與學歷條件，即可快速計算出 116 年調薪後的具體金額。
* **學制與條件篩選：** 支援不同教育階段（國小、國中、高中職等）與職務加給的彈性調整。
* **清晰的數據視覺化：** 透過直觀的介面，呈現調薪前後的增幅比例與結構變化。

## 📱 使用說明
1. 開啟線上展示連結。
2. 於下拉選單中選擇您的**現行學歷**與**薪級**。
3. 勾選是否有兼任導師或行政職務（計算職務加給）。
4. 點擊「開始試算」，系統即刻呈現 116 年新制薪資與調薪前後差額。

## 🛠️ 技術棧
* **前端網頁：** HTML5 / CSS3 / JavaScript (Vanilla JS)
* **資料處理：** 整合 116 年公立學校教師薪給表之公開數據進行精確計算。
